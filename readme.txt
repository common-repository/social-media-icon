== Description ==

Social Media is a wordpress icon generator plugin. Create social icons. It is easy to use and easy to customize. You can create social group of buttons. We have 10 Themes of Icons with live preview. Create social icons in widget.

Using [Social_Media id=XX] shortcode, you can publish in any Page or Post in your WordPress sites.

= Requirements = 
* PHP 5 or later
* Wordpress installed

== Installation ==

1. Upload the entire **social-media** folder to the **/wp-content/plugins/** directory.
2. Activate the plugin through the **Plugins** menu in WordPress admin.
3. Add new group, Use **[Social_Media id=652]** shortcode to publish your Icon into any page or post.
